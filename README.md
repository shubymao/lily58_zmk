# Lily58 ZMK Config

ZMK user config for a split Lily58 using `nice_nano` controllers.

## What this repo builds

- `lily58_left` firmware
- `lily58_right` firmware

## CI/CD

GitHub Actions builds firmware on:

- `push`
- `pull_request`
- manual `workflow_dispatch`

Download UF2 artifacts from the workflow run in the Actions tab.
